# FitnessTrack

## FitnessTrack
* Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
* Käyttäjä pystyy pystyy tallentamaan painonosto tulokset
* Käyttäjä pystyy katsomaan läpi oman historian
* Käyttäjä pystyy jakamaan omat tulokset valituille käyttäjille
* käyttäjät pystyvät luomaan ryhmiä joiden sisällä tulokset jaetaan automaattisesti kaikkien kanssa
